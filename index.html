Phase 1: Basic JavaScript Projects

এই ফেইজে আমরা জাভাস্ক্রিপ্ট এর একদম বেসিক জিনিসগুলো যেমন DOM ম্যানিপুলেশন, ইভেন্ট লিসেনার নিয়ে কাজ করবো।

1. Counter App

Project #1

একটি সিম্পল কাউন্টার যেখানে increment, decrement এবং reset বাটন থাকবে।

What you'll learn (কী শিখবেন):

DOM Manipulation Event Listeners State Update

2. Digital Clock

Project #2

লাইভ সময় দেখানোর জন্য একটি ডিজিটাল ঘড়ি।

What you'll learn (কী শিখবেন):

Date API setInterval() Time Formatting

3. Random Quote Generator

Project #3

বাটনে ক্লিক করলে নতুন নতুন উক্তি (Quote) দেখাবে।

What you'll learn (কী শিখবেন):

API Calls (fetch) async/await JSON Parsing

4. Character Counter

Project #4

টেক্সটবক্সে কতগুলো অক্ষর লেখা হয়েছে তা লাইভ কাউন্ট করবে।

What you'll learn (কী শিখবেন):

Textarea Events String Manipulation Live Feedback

Phase 2: Intermediate JS Projects

এখন আমরা ডেটা সেভ করা, এপিআই নিয়ে আরেকটু গভীরে কাজ করা এবং ডাইনামিক ফিল্টারিং শিখবো।

5. Todo App

Project #5

কাজ লিস্ট করার অ্যাপ যেখানে টাস্ক অ্যাড, ডিলিট এবং কমপ্লিট করা যাবে।

Features & Skills (ফিচার ও স্কিল):

CRUD Operations localStorage Data Persistence

6. Weather App

Project #6

শহরের নাম দিলে সেখানকার বর্তমান আবহাওয়া দেখাবে।

Features & Skills (ফিচার ও স্কিল):

External REST APIs Advanced async JS Error Handling

7. Search Filter System

Project #7

লিস্টের মধ্যে কিছু সার্চ করলে সাথে সাথে ফিল্টার হয়ে শুধু ম্যাচ করা ডেটা দেখাবে।

Features & Skills (ফিচার ও স্কিল):

Array.filter() Dynamic DOM Updates Search Inputs

8. Image Slider

Project #8

ছবি ডানে-বামে স্লাইড করে দেখার সিস্টেম (ক্যারোজেল)।

Features & Skills (ফিচার ও স্কিল):

Advanced DOM CSS Transitions JS Array Index Tracking

Phase 3: Algorithm + Logic Projects

এই ফেইজে আপনার লজিক বিল্ডিং স্ট্রং হবে। ডেটা কীভাবে কাজ করে, কীভাবে খুঁজবে বা সাজাবে তা শিখবেন।

9. Sorting Visualizer

Project #9

অ্যালগরিদম কীভাবে কাজ করে তার ভিজ্যুয়াল উপস্থাপন (এনিমেশন)।

Features & Skills (ফিচার ও স্কিল):

Bubble Sort Selection Sort Timing/Delays in JS

10. Advanced Search System

Project #10

বড় ডেটাবেজ থেকে দ্রুত তথ্য খুঁজে বের করার সিস্টেম।

Features & Skills (ফিচার ও স্কিল):

Binary Search Dynamic Searching Performance

11. Pagination System

Project #11

অনেক ডেটাকে ছোট ছোট পেইজে ভাগ করে দেখানো (১, ২, ৩... পেইজ)।

Features & Skills (ফিচার ও স্কিল):

Next/Prev Logic API Data Pagination Math/Calculations

12. Infinite Scroll

Project #12

পেইজের নিচে গেলে অটোমেটিক নতুন ডেটা লোড হবে (ফেসবুক/ইন্সটাগ্রাম এর মত)।

Features & Skills (ফিচার ও স্কিল):

Lazy Loading Scroll Events Intersection Observer

Phase 4: Real World Projects

এখন আমরা প্রফেশনাল প্রজেক্ট বানাবো যেগুলো পোর্টফোলিওতে রাখার মত এবং ক্লায়েন্টরা সচরাচর চেয়ে থাকে।

13. Blog Website

Project #13

পোস্ট লিস্ট, ক্যাটাগরি, সার্চ এবং ফিল্টার সহ পূর্ণাঙ্গ ব্লগ।

Features & Skills (ফিচার ও স্কিল):

Post Rendering Search + Filter Routing Logic

14. E-commerce Product Filter

Project #14

ই-কমার্স সাইটের মত প্রাইস, ক্যাটাগরি অনুযায়ী প্রোডাক্ট ফিল্টার করা।

Features & Skills (ফিচার ও স্কিল):

Multi-level Filtering Sorting (Price/Name) Complex Data Handling

15. Dashboard UI

Project #15

চার্ট এবং ডাইনামিক ডেটা দিয়ে এডমিন প্যানেল ড্যাশবোর্ড তৈরি।

Features & Skills (ফিচার ও স্কিল):

Chart JS / APIS Dashboard Layouts Dynamic Statistics

16. Advanced Task Manager

Project #16

প্রায়োরিটি, ডেডলাইন, ড্র্যাগ এন্ড ড্রপ সহ প্রজেক্ট ম্যানেজমেন্ট টুল (Trello Clone)।

Features & Skills (ফিচার ও স্কিল):

Priority Management Deadlines/Dates Complex State

Phase 5: Advanced Projects

ফাইন্যাল স্টেপ! এখানে আমরা পারফরম্যান্স অপ্টিমাইজেশন এবং অ্যাডভান্সড ফিচার নিয়ে কাজ করবো।

17. Real-time Chat UI

Project #17

লাইভ মেসেজিং ইন্টারফেস, টাইপিং ইন্ডিকেটর এবং রিয়েল টাইম ডেটা ফ্লো।

Features & Skills (ফিচার ও স্কিল):

Chat UI Patterns Typing Effects WebSockets (Basic)

18. Performance Optimized Blog

Project #18

কোড স্প্লিটিং, লেজি লোডিং ব্যাবহার করে অত্যন্ত ফাস্ট একটি ব্লগ তৈরি।

Features & Skills (ফিচার ও স্কিল):

Code Splitting Advanced Lazy Loading Performance Tuning

19. Full API Dashboard

Project #19

একাধিক API থেকে ডেটা এনে একসাথে প্রসেস করে ভিজ্যুয়ালাইজ করা।

Features & Skills (ফিচার ও স্কিল):

Multiple API Handling Data Visualization Promise.all()

20. Developer Portfolio

Project #20

আপনার নিজের কাজের শোকেস করার জন্য কন্ট্যাক্ট ফর্ম সহ একটি প্রফেশনাল পোর্টফোলিও।

Features & Skills (ফিচার ও স্কিল):

Project Showcase Form Validation Personal Branding

🎯 Ultimate Result (ফলাফল)

JavaScript Strong Foundation DOM Master API Expert Performance Optimization Problem Solving Skills
