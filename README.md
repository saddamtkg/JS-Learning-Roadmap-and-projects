# JavaScript Pro Roadmap — শেখার সাইট

এই রিপোজিটরিতে **১২ মাসের JavaScript + DSA ও Web Optimization** রোডম্যাপ আছে। বাংলা ও ইংরেজিতে ব্যাখ্যা, ২০টি প্রজেক্ট, সিলেবাস ও ডকুমেন্টেশন সহ একটি সম্পূর্ণ শেখার সাইট।

---

## 🚀 সাইট লাইভ করতে (GitHub Pages)

### ধাপ ১: রিপো আপলোড করুন
১. GitHub এ নতুন Repository তৈরি করুন। (নাম যেমন: `js-mastery-roadmap`)  
২. **নিচের ফাইল/ফোল্ডারগুলো** রিপোতে আপলোড করুন:

| প্রয়োজনীয় | বর্ণনা |
|------------|--------|
| `index.html` | মেইন পেজ (রোডম্যাপ, সিলেবাস, প্রজেক্ট) |
| `docs.html` | ডকুমেন্টেশন পেজ (সব টপিকের বিস্তারিত) |
| `main.js` | সাইটের লজিক ও ডেটা লোডিং |
| `docs.js` | ডক পেজ: সাইডবার অ্যাকটিভ সেকশন + কোড কপি বাটন |
| `docs.css` | ডক পেজ স্টাইল |
| `animations.css` | স্ক্রল অ্যানিমেশন ও UI স্টাইল |
| `scroll-animations.js` | স্ক্রল ট্রিগার অ্যানিমেশন |
| `site-optimizer.js` | পারফরম্যান্স উইজেট (ডেট + ক্লক) |
| `data/` ফোল্ডার | `tasks.json` + `projects.json` (সিলেবাস ও প্রজেক্ট ডেটা) |

> ⚠️ **জরুরি:** `data` ফোল্ডার এবং এর ভেতরের দুটো JSON ফাইল অবশ্যই আপলোড করতে হবে। এগুলো ছাড়া সাইটে সিলেবাস ও প্রজেক্ট লোড হবে না।

### ধাপ ২: GitHub Pages চালু করুন
১. Repository এর **Settings** এ যান।  
২. বাম মেনু থেকে **Pages** ক্লিক করুন।  
৩. **Build and deployment** → **Source**: `Deploy from a branch` সিলেক্ট করুন।  
৪. **Branch**: `main` (বা `master`) এবং `/ (root)` সিলেক্ট করে **Save** করুন।  
৫. কয়েক মিনিট পর সাইট লাইভ হবে। লিংক পাবেন: `https://<your-username>.github.io/<repo-name>/`

### ধাপ ৩: চেক করুন
- হোমপেজে প্রজেক্ট গ্রিড ও সিলেবাস ট্যাব দেখাচ্ছে কিনা।  
- **View Task** ক্লিক করলে মডাল খুলছে কিনা।  
- **Documentation** লিংক থেকে `docs.html` ওপেন হচ্ছে কিনা।

## 📁 সাইট স্ট্রাকচার (লাইভের জন্য যা লাগে)

```
My Programming-roadMap/
├── index.html          ← মেইন পেজ
├── docs.html           ← ডকুমেন্টেশন
├── main.js             ← অ্যাপ লজিক
├── docs.js             ← ডক পেজ (সাইডবার + কপি বাটন)
├── docs.css            ← ডক পেজ স্টাইল
├── animations.css      ← অ্যানিমেশন স্টাইল
├── scroll-animations.js
├── site-optimizer.js   ← অপ্টিমাইজার উইজেট + ক্লক
├── data/
│   ├── tasks.json      ← সিলেবাস টাস্ক (View Task এর ডেটা)
│   └── projects.json  ← ২০ প্রজেক্টের তালিকা
└── README.md
```

`my file` ফোল্ডার (এক্সেল/টেক্সট সোর্স) সাইট চালানোর জন্য **দরকার নেই** — সব কনটেন্ট ইতিমধ্যে `data/*.json` এ আছে। চাইলে এক্সেল বন্ধ করে ফোল্ডার ডিলিট করতে পারেন।
